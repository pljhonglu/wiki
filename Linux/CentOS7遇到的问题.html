关闭 selinux

遇到一个坑，selinux 开启状态导致 nginx 无法访问目录，即使 nginx 有目录权限也不行。后来找到原因是 selinux 的锅。可以通过下面方法关闭 selinux。

查看 selinux 状态

# 查看 selinux 状态
selinuxenabled && echo enabled || echo disabled

# 关闭 selinux
nano /etc/sysconfig/selinux
SELINUX=disabled

# 重启
reboot
