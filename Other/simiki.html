安装

pip install simiki --user

安装到 ~/Library/Python/2.7/bin 目录下

添加到环境变量

打开 ~/.zshrc 添加如下行

export PATH=$HOME/Library/Python/2.7/bin:$PATH

安装过程中出现的问题

raise UnknownTimeZoneError(zone)
...
UnknownTimeZoneError: (UnknownTimeZoneError('local',), <function _p at 0xb676db54>, ('local', 28800, 0, 'CST'))

因为系统中缺少 timezone 文件，如下创建

vi /etc/timezone

timezone 文件内容

Asia/Shanghai
