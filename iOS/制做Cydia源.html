制作 Packages

生成好的 deb 包放到 www/Cydia/debs/ 目录下，执行命令：

dpkg-scanpackages debs /dev/null > Packages
tar zcvf Packages.gz Packages
bzip2 -k Packages Packages.bz2

这样 Package 、Packages.bz2、Package.gz就有了

制作 Release

Release 文件内容：

Origin: Cydia/NetEase
Label: Cydia/NetEase
Suite: stable
Version: 1.0
Codename: netease
Architectures: iphoneos-arm
Components: main
Description: Distribution of Unix Software for iOS

说明

如果不签名，那么就不需要 md5sum 相关的信息
